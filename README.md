# ShaderPlayground
Playground for Metal Shading Language

MSL(Metal Shading Language)で遊べるサンプルプログラムです。
iPhone上でMSLをコンパイル＆実行できます。

入力はピクセル座標の他に
* 解像度
* 経過時間
* マイク音量
* 加速度センサー
を利用できます。
